Current Updates:

April 11th, 2024:

The backend Flask server has been created. It has been written to retrieve the list of all current accounts.

The code for creating accounts has been written, but the REST api call from the front-end to the backend
is currently in the works

Today, Angular has officially been decided as the front-end framework. The reasoning behind choosing
Angular over React is due to the better structure and organization of Angular. The project will be
easier to maintain and scale with Angular. This is intended to be a longer term project. Using Angular
will make it easier to keep and maintain the project, especially when adding new features and connecting
to the backend.
